📋 ¿Cómo Funciona el Proceso?

🕷️ Paso 1: Web Scraping

Extrae precios de 5 tiendas online: Celudmovil, Clevercel, iTech, PhoneElectric y Tooho.

🔧 Paso 2: Normalización

Limpia y estandariza los datos, filtrando solo productos NUEVOS y SEMINUEVOS.

☁️ Paso 3: Subida a SharePoint

Sube los archivos normalizados a SharePoint, sobrescribiendo los anteriores.

📊 Paso 4: Generación de Excel

n8n procesa los datos y genera el Excel de comparación de precios.

Flujo Detallado

  1. Scraping de Tiendas
    Se ejecutan 5 spiders de Scrapy que extraen productos de cada tienda (~4-5 min)
  2. Normalización de Datos
    Se limpian nombres, detectan marcas, extraen especificaciones y filtran condiciones (~1-2 min)
  3. Subida a SharePoint
    Se suben 5 archivos JSON a SharePoint/results_normalized/ (~1 min)
  4. Trigger de n8n
    Se notifica a n8n para que procese los archivos y genere el Excel (~2-3 min)
  5. Excel Actualizado
    El archivo results_comparison_complete.xlsx está disponible en SharePoint
✨ Mejora Implementada: Los archivos ahora se sobrescriben en lugar de acumularse. Esto significa que SharePoint siempre tendrá solo los 5 archivos más recientes (uno por tienda), evitando la acumulación de versiones antiguas y asegurando que el Excel solo contenga precios actuales.
